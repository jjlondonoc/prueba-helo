# Prueba técnica para Helo

**Autor:** Juan José Londoño - Ingeniero Biomédico M.Sc

**Fecha**: 30/11/2025

## Descripción

En este repo se encuentran las 6 carpetas con las soluciones a las 6 actividades planteadas por la prueba técnica para Helo. Se encuentra en algunas carpetas un README con explicación adicional de lo que hice, los patrones de diseño que apliqué y las prácticas que usé.

El diseño de PCB lo realicé con **kicad**, configurando el entorno y especificaciones de la tarjeta alineado con el fabricante JLCPB, admeás seleccioné los componentes específicos según criterios de diseño como consumo, corrientes, rendimiento, etc. y generé el BOM alineado con JLCPB.

> Quedo atento a cualquier duda sobre las soluciones a las actividades en esta prueba técnica.
